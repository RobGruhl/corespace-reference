Round Structure Overview

Each round consists of 5 phases in strict order. All characters act each round, then the round repeats until mission ends.

1. HOSTILITY PHASE - Danger escalates
2. TRADER PHASE - Players take actions
3. FIRST BORN PHASE - Enemies arrive & attack
4. NPC PHASE - Other characters activate
5. ASSESSMENT PHASE - Clean up & reset
‚Üª REPEAT UNTIL MISSION ENDS

1. Hostility Phase

Purpose: Escalate danger, introduce random events

Step 1: Add Hostility Peg

  • Solo / 2-3 crews: Add 1 black peg
  • 4-6 crews: Add 2 black pegs (1 per 3 crews, rounded up)
  • Place in first empty space on tracker

Step 2: Draw Event Card

  • First player draws from event deck
  • Look at current hostility level on tracker
  • Resolve effects for that level ONLY

Step 3: Resolve Event

Assistance Card (üëç)?
  • Keep it secret (don't show other players)
  • Can use later during your turn
  • Provides special benefit
Can't Fully Resolve?
  • Resolve as much as possible
  • If completely unresolvable + has NPC icon: Place NPC instead
  • Shuffle card back into deck
  • Draw another card

Step 4: Discard Card

  • Place in faceup discard pile
  • If deck runs out: Shuffle discards to make new deck

2. Trader Phase

Purpose: Players control their traders

Turn Order

  1. Player with turn counter goes first
  2. Proceed clockwise around table
  3. Each player activates ONE trader
  4. Continue until all traders activated

Activating a Trader

Choose unactivated trader (no activation counter)
Declare which trader you're activating
Take actions (number = ACTION stat)
Place activation counter on dashboard
Pass to next player clockwise

Available Actions

See Actions page for complete list. Common actions:

Special Cases

Prone Characters: Can only Stand Up (then act normally)
Defeated Characters: Cannot act at all (need revival)

3. First Born Phase

Purpose: Enemies arrive and attack using AI

Part 1: Arrival

Check current hostility level
For each icon in that level (top to bottom):
Number shown = that many arrive
Die icon = roll die for quantity
Roll chance die for entry point
Place in contact with entry point
Rank 4+ First Born: Once per game only (Live One, Gatherer, Annihilator, True Born)
At CLEANSE: Use "Charged" side of First Born board
No units available?: Use next lowest rank

Part 2: Activation

Order: Activate all First Born, highest rank first

Targeting (Before Activation)

Closest non-First Born in LoS, not in cover
If all in cover: Closest visible
If none visible: Closest (measure around walls)
Exception - Assassins: Always target Captain if visible
Target LOCKS for entire activation (unless defeated)

AI Decision Tree (Each Action)

Target engaged? ‚Üí Close Assault
Target not in LoS? ‚Üí Move toward target
No shooting stat? ‚Üí Move toward target
Can reach & attack this turn? ‚Üí Move toward target
Already moved? ‚Üí Ranged Assault
Shooting > Close stat? ‚Üí Ranged Assault
Otherwise ‚Üí Move toward target
First Born Special Rules:
  • Move as far as possible (stop if reach target)
  • Ranged assault: UNLIMITED range
  • Unaffected by üí• results
  • Knocked prone: Must Stand Up first action

Knowledge Die

Drones and some First Born roll Knowledge Die after attack:

  • Apply special effect based on result
  • When CLEANSE: May use multiple times per round (icon shows "2")

4. NPC Phase

Purpose: Other characters (non-First Born) activate

Activation Order

  1. Game Hunters (if present)
  2. Other NPC factions from optional sets

Within each type: Highest rank acts first

Arrival

Game Hunters and Rock Worms arrive via Event cards. Place them at the entry point or worm hole indicated by the card.

NPC Behavior

Each NPC type has specific rules (see Enemies page). In First Born the default NPCs are Game Hunters:

  • Target First Born ahead of other threats before pursuing Beasts
  • Move up to 4" per action, choosing the safest path toward their quarry
  • Collect Trophy tokens from fallen Beasts and withdraw once burdened
  • Are unaffected by üí• results and avoid hazards unless no other route exists
  • Knocked prone: Stand up before taking further actions

Joining Crews

If Persuaded or by Event:
  • NPC joins crew for rest of mission
  • Activated in Trader Phase by that player
  • Keep NPC stats (don't flip board)
  • If defeated: Removed, cannot revive

5. Assessment Phase

Purpose: Clean up and prepare for next round

Steps

  1. Remove all activation counters
  2. Resolve end-of-round effects
    • Any abilities triggering "at end of round"
    • Rotate timed equipment tokens
    • Check mission-specific conditions
  3. Pass turn counter clockwise
    • Next player becomes first player
    • They go first in next round's Hostility Phase
That's it! Start next round with Hostility Phase

Mission End Conditions

Mission ends immediately when:

  • All traders defeated, OR
  • All traders back on ships (exited board)

Winning

  • If entire crew defeated: Automatic loss
  • If primary objective completed: Winner
  • If nobody completed: All players lost

Tiebreakers (Multiplayer)

  1. Most traders alive
  2. Most valuable equipment (token sell values)
