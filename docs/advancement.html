Skills Overview

Cost: Skill pegs = level chosen (1-3)
Limit: ONE skill use per round (or spend 1 peg for 2nd effortless action)
Permanent Cost: Spent pegs stay removed entire game

Skill Types

Always on, no peg cost
Active
Use during your turn

Campaign System

Post-Game Sequence

  1. Extraction Phase - Rescue defeated traders
  2. Advancement Phase - Gain career points
  3. Trade Phase - Buy/sell equipment
  4. Maintenance Phase - Repair ship

Extraction Phase

If all traders made it out: Skip this phase

For each defeated trader:

  1. Roll chance die
  2. 💀 result = Permanently dead (remove from campaign)
  3. Otherwise, choose option:
    • Rescue Mission: Play immediate mission on same board
    • Emergency Teleport: Roll on table for result
    • On Your Own: Roll on table if no crew aboard

Advancement Phase

Career Points: Each trader that survived AND made it to ship gains +1 career point
True Born Kill: Exchange counter for +1 additional point

Spending Career Points

  • Fill next empty space on trader board
  • Can also allocate to skills on class board

Leveling Up

LevelRewards
Level 1+1 Health, +1 Skill (permanent)
Level 2+1 Health, +1 Skill (permanent)
Level 3+1 Health, +1 Skill, +1 Action (permanent, max level)

Trade Phase

Each Crew Chooses ONE Trading Post

  • Buy items: Pay cost (first number on token)
  • Sell items: Receive value (second number on token)
  • Rare items (💎): Cannot be bought
  • Objective tokens with UA: Can be sold
  • Stock limited to available tokens
  • Can barter with other players

Hiring Crew

  • Can hire new traders (max 7 total)
  • Pay cost shown

Permanent NPC Hires

If NPC joined crew during mission AND made it to ship:
  • Can permanently hire for cost shown
  • Becomes trader for future missions
  • Exception: Galactic Corps and Security cannot hire

Maintenance Phase

Ship Systems

  1. Degradation Check: Roll chance die, apply to ship system
  2. Repairs: 2 UA per repair point (fill 1 space)
  3. Upgrades: Purchase ship upgrades (Scanners, Auto-Defences, Docking Thrusters - 3 levels each)
If any system reaches 0: Ship destroyed, crew out of campaign (unless repair this phase)
